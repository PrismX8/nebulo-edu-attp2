Nebulo OS Help

Step-by-step reminders on keeping your workspace moving smoothly.

Launching Apps

  • Click an icon on the desktop to open it in a new window. Double-clicking brings it to focus.
  • Use the taskbar at the bottom to reopen or minimize any running window.
  • The start menu mirrors the same listâ€”tap a card to launch the corresponding app.

Window Controls

  • The window header contains minimize, maximize, and close controls. Hover to see the tooltips.
  • You can move windows by dragging the header and resize them from the edges or corners.
  • If something blocks a window, you can refresh the entire desktop with the tray refresh button.

Organizing Icons

  • Drag icons on the desktop to rearrange them. If you accidentally start dragging, just release and click again.
  • Position data is saved automatically so your layout stays consistent across sessions.
  • Right-click an icon for options like pinning it to the taskbar for quick access.

Settings & Preferences

  • Open Settings to switch themes, set a panic key, or update your background image.
  • Any changes dispatch live events so running apps can react to fresh preferences.
  • Export/Import lets you back up cookies, localStorage, and theme settings for safekeeping.
